openEO Viewer

Open EO Viewer is a lightweight Leaflet application for exploring NASA's Global Imagery Browse Services (GIBS) layers. Switch between true-color products, pick a date, and sketch areas of interest directly on the map.

Features

  • Leaflet + GIBS integration – Browse daily NASA imagery without an API key
  • Layer switching – Toggle between VIIRS and MODIS true-color layers
  • Date selection – Navigate through historical imagery
  • Drawing tools – Capture polygons or rectangles with live measurements
  • Export options – Save as GeoJSON, KML, PNG, or GeoTIFF
  • Storm awareness – Visualize current severe storms from NASA EONET

About This Project

This codebase is part of a pedagogical exercise for International Space University's MSS26 class, designed to teach students software engineering fundamentals using AI assistance and GitHub collaboration.
